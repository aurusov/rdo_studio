Cygwin используется для компиляции некоторых проектов, например ICU

1. Скачать setup.exe с сайта
http://www.cygwin.com/

Прямая ссылка
http://cygwin.com/setup.exe

2. Запустить setup.exe
2.1 Install from Internet, выбрать любой сервер, выбрать пути
2.2 Выделить проекты для установки
make
autoconf

Их можно найти поиском, лежат они в разделе Devel
