Установка из бинарников требует четкого понимания: если версия компилятора, на котором были собраны бинарники, отличается от вашего компилятора, то это вариант установки не подходит
Текущие бинарники собраны на
Visual Studio 2008 + SP1. Полная версия 9.0.30729.1 SP
Загляните в About Visual Studio и сверьтесь

1. Загрузка исходников Qt
http://qt-project.org/downloads

Ищем ссылку в виде:
The source code is available as a zip (270 MB)

Архив должен называться
qt-everywhere-opensource-src-<версия>.zip

Качаем, распаковываем в любое место, например, в C:\qt-5.0.1. Эта директория должна содержать
gnuwin32
qtactiveqt
qtbase
qtdeclarative
qtdoc
qtgraphicaleffects
qtimageformats
qtjsbackend
qtmultimedia
и др. файлы и папки

2. Загрузка бинарников
Открываем папку из google-docs
https://docs.google.com/folder/d/0B-tnm4ER5l74c0hnRFl4LTVHbDA/edit?usp=sharing
и выкачиваем архивы
qt-5.0.1-qbase-bin-vs2008-sp1.7z
qt-5.0.1-qbase-include.7z
qt-5.0.1-qbase-lib-vs2008-sp1.7z
qt-5.0.1-qbase-plugins-vs2008-sp1.7z
qt-5.0.1-qbase-src-corelib-global.7Z

Их содержимое необходимо расположить соответственно в папки
C:\qt-5.0.1\qtbase\bin
C:\qt-5.0.1\qtbase\include
C:\qt-5.0.1\qtbase\lib
C:\qt-5.0.1\qtbase\plugins
C:\qt-5.0.1\qtbase\src\corelib\global

3. Настройка окружения и IDE

3.1. Добавляем переменную окружения
QTDIR=C:\qt-5.0.1\qtbase
Не забывам перелогиниться в винде.

3.2. Для Windows запускаем Visual Studio, заходим в настройки
Tools/Options/Projects and Solutions/VC++ Directories

Platform             = Win32
Show directories for = Include files, добавляем $(QTDIR)\include
Show directories for = Library files, добавляем $(QTDIR)\lib
