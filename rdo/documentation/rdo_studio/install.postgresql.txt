PostgreSQL - ядро базы данных, с которым работает РДО.


1. Загрузка инсталятора
http://www.enterprisedb.com/products-services-training/pgdownload

Находим и загружаем последнюю стабильную версию PostgreSQL x32. На текущий момент это
Version 9.2.2


2. Установка
Устанавливаем в C:\psql
Оставляем все предложеные параметры со значениями по умолчанию, кроме одного: пароль к учетной записи "rdo"
Stack builder (в конце установки) нам не нужен

3. Настройка окружения
3.1. Добавляем переменную окружения PSQL

Находим директорию, куда поставили PostgreSQL, её содержимое выглядит как:
bin\
data\
doc\
include\
и т.д.

И прописываем PSQL
PSQL=C:\psql

в Win 7/8 это делается так:
Панель управления -> Учетные записи пользователей -> Изменение переменных среды -> создаём PSQL и прописываем значение

Не забывам перелогиниться в винде

4. Добавить либы необходимые для сборки qt-драйвера для PostgreSQL
4.1. Открываем папку из google-docs
https://docs.google.com/folder/d/0B-tnm4ER5l74c0hnRFl4LTVHbDA/edit?usp=sharing
и выкачиваем архив
zlib.7z

Их содержимое необходимо расположить в папки:
zdll.lib :   C:\Program Files (x86)\Microsoft Visual Studio 9.0\VC\lib
zlib1.dll:   C:\qt-5.0.1\qtbase\bin
