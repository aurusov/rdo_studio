Инструкции по сборке РДО и проведению тестов.

1. Генерация Makefile'ов

Открываем консоль. Для Windows
Пуск\Microsoft Visual Studio 2012\Visual Studio Tools\VS2012 x86 Native Tools Command Prompt

cd <корень_проекта> (например, C:/rdo/rdo_studio)
mkdir build
cd build
cmake ..

2. Компиляция
2.1 Для Windows
В папке build находим rdo.sln. Жмакаем по нему дважды или открываем из Visual Studio. Выбираем конфигурацию Debug. Компилируем проект rdo_studio:
.apps/.rdo_studio/rdo_studio, popup-menu, Set as StartUp Project, Build

2.2 Для Windows с использованием gcc через MinGW
Пути к C:\MinGW\bin и C:\MinGW\msys\1.0\bin должны быть в PATH
Запускаем 'MinGW Shell', окрывается консоль
cd /c/rdo/dev_linux
mkdir build
cd build
cmake -G 'MinGW Makefiles' ..
cmake --build ./

Если возникла ошибка при "cmake -G 'MinGW Makefiles' ..", то еще раз повторяем, у меня срабатывает со второго раза

2.3 Для Linux
make -j4

3. Запуск тестов для Linux

cd <корень_проекта>
cd build
make test
