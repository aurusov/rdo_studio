svn -  централизованная система управления версиями (http://ru.wikipedia.org/wiki/Subversion)

1. Для Windows
1.1. Скачать и установить TortoiseSVN
    http://tortoisesvn.net/downloads.html
	
	ВАЖНО! В момент установки жмакните напротив 'установить command line client tools'

1.2 Путь к svn.exe должен быть в PATH. Напишите сюда, делает ли это инсталятор автоматически.

2. Для GNU/Linux
    Установить из репозиториев своего дистрибутива, например, для Debian/Ubuntu:

apt-get install svn
