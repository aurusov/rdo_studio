ОБЪЕКТ СЦЕНАРИЯ ЗАПУСКОВ

Объект сценария запусков (с расширением .smr) описывает сценарий запусков данной модели состоит из одного или нескольких <блоков_прогона>.

Сценарий запуска имеет следующий формат:

<серия_прогонов> | <единичный_прогон>
серия_прогонов
Серия прогонов описывается следующим образом: 
{ <один_прогон_из_множества> }
один_прогон_из_множества
Один прогон внутри серии имеет следующий формат: 
'{' <единичный_прогон> '}'
единичный_прогон
Один прогон описывается следующим образом: 
{ <инструкция_настраивающая_прогон> }
инструкция_настраивающая_прогон
Инструкция для настройки прогона настраивает режимы моделирования, новые значения констант, баз генераторов, параметров ресурсов, точки останова, терминальное условие и прочее: 
<инструкция_окончания_прогона>
| <инструкция_планирования_события>
| <точка_останова>
| <время_начала_прогона>
| <время_начала_трассировки>
| <время_окончания_трассировки>
| <номер_кадра>
| <новое_значение_константы>
| <инструкция_нового_значения_параметра_ресурса>
| <инструкция_нового_значения_базы_генератора>
| <масштаб_отображения>
| <режим_отображения>
инструкция_окончания_прогона
Инструкция окончания прогона имеет следующий формат: 
Terminate_if <условие_остановки>
условие_остановки
Условие остановки прогона - логическое выражение. Прогон будет автоматически завершен в тот момент, когда значением указанного логического выражения станет ИСТИНА. Если условие окончания прогона не задано, автоматического завершения прогона не происходит.
инструкция_планирования_события
Инструкция планирования события имеет формат: 
<имя_события>.planning( <время_планирования_события> [, <значение_параметров_события> ] );
имя_события
Имя события - это имя одного из описанных в модели событий.
время_планирования_события
Время планирования события представляет собой арифметическое выражение, равное времени, в которое данное событие выполнится.
значение_параметров_события
Значения параметров события записываются таким же образом, как и для активностей, за исключением того, что они могут представлять собой арифметическое выражение.
точка_останова
Задание точки останова имеет следующий формат: 
Break_point <имя_точки_останова> <условие_остановки>
имя_точки_останова
Имя точки останова - это простое имя, оно не должно совпадать с ранее определенными именами.
условие_остановки
Логическое выражение задает условие, которому должно удовлетворять состояние системы. В тот момент, когда указанное логическое выражение приобретает значение ИСТИНА, происходит прерывание моделирования с выдачей соответствующей информации и РДО-имитатор переходит в режим монитора.
время_начала_прогона
Задание времени начала прогона имеет следующий формат: 
Run_StartTime = <константа>
константа
Вещественная константа, задающая модельное время, устанавливаемое в начале прогона. Значение по умолчанию - 0.0.
время_начала_трассировки
Задание времени начала трассировки имеет следующий формат: 
Trace_StartTime = <константа>
константа
Вещественная константа, задающая время начала выдачи информации в Объект трассировки. Значение по умолчанию - 0.0.
время_окончания_трассировки
Задание времени окончания трассировки имеет следующий формат: 
Trace_EndTime = <константа>
константа
Вещественная константа, задающая время окончания выдачи информации в Объект трассировки. Значение по умолчанию - 10.0Е+30.
номер_кадра
Задание номера кадра имеет следующий формат: 
Frame_number = <константа>
константа
Константа целого типа, задающая номер кадра из числа описанных в объекте кадров для отображения при запуске модели, если указан соответствующий режим. Значение по умолчанию равно 1.
новое_значение_константы
Задание нового значения константы имеет следующий формат: 
<имя_константы> = <константа>
имя_константы
Имя константы - это одно из имен констант, описанных в объекте констант.
константа
Новое значение символьной константы задается целой или вещественной численной константой либо именем значения в соответствии с типом константы.
инструкция_нового_значения_параметра_ресурса
Инструкция нового значения параметра ресурса имеет следующий формат: 
<имя_ресурса>.<имя_параметра> = <константа>
имя_ресурса
Имя ресурса - это одно из имен ресурсов, заданное в объекте ресурсов.
имя_параметра
Имя параметра - одно из имен параметров, описанных в объекте типов ресурсов для соответствующего типа.
константа
Новое значение символьной константы задается целой или вещественной численной константой либо именем значения в соответствии с типом константы.
инструкция_нового_значения_базы_генератора
Инструкция нового значения базы генератора имеет следующий формат: 
<имя_последовательности>.Seed = <константа>
имя_последовательности
Имя последовательности - это одно из имен, указанных при описании последовательностей.
константа
Значение задают численной константой целого типа в диапазоне -2147483647…2147483647.
масштаб_отображения
Задание масштаба отображения имеет следующий формат: 
Show_rate = <константа>
константа
Задает временной масштаб отображения.Вещественная константа, равная отношению темпа отображения к модельному времени. Например, при Show_rate = 60.0 отображение будет производиться в 60 раз быстрее, т.е. один час модельного времени будет отображаться за одну минуту. Значение по умолчанию - 60.0.
режим_отображения
Задание режима отображения имеет следующий формат: 
Show_mode = <режим>
режим
В данной версии языка возможны режимы следующих типов:
Режим Описание
NoShow Режим без отображения
Monitor Режим монитора
Animation Режим отображения состояния (анимации)
Значение по умолчанию - NoShow.

Пример задания одного прогона

 
	Show_mode = NoShow
	Show_rate = 3600.0

	Terminate_if Ресурс_2.Параметр_5 = Занят 

	Константа_1 = 20
	Константа_5 = Занят

	Ресурс_1.Параметр_2 = 3.4
	Ресурс_3.Параметр_5 = Разгрузка

	Последовательность_4.Seed = 363252

Пример задания серии прогонов

 
{
	Show_mode      = NoShow
	Show_rate      = 3600.0
	Образец_прихода_клиента.planning( time_now )
	Интервал_прихода.seed = 987654323
	Константа_1 = 20
	Terminate_if Time_now >= 12 * 7 * 60
}

{
	Show_mode      = NoShow
	Show_rate      = 3600.0
	Образец_прихода_клиента.planning( time_now )
	Интервал_прихода.seed = 123549
	Ресурс_3.Параметр_5 = Разгрузка
	Terminate_if Time_now >= 12 * 7 * 60
}
