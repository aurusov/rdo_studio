СПЕЦИАЛЬНЫЕ СИМВОЛЫ

В данной версии языка лексемами являются следующие специальные символы и их комбинации:

Комбинация Значение
: Двоеточие
[, ] Прямые скобки
(, ) Kруглые скобки
{, } Фигурные скобки
, Запятая
+ Знак сложения
- Знак вычитания или унарного минуса в зависимости от ситуации
* Знак умножения
/ Знак деления
< Знак меньше
> Знак больше
= Знак равно
<> Не равно
<= Меньше или равно
>= Больше или равно
! Знак отрицания
.. Знак диапазона
