АЛФАВИТ

Алфавит языка содержит следующие символы:

  • цифры 0…9
  • прописные и строчные латинские буквы A…Z, a…z
  • прописные и строчные русские буквы А…Я, а…я
  • специальные символы:
    • Символ Значение
    _ подстрочный символ
    $ знак денежной единицы
    : двоеточие
    [ ] прямые скобки
    ( ) круглые скобки
    { } фигурные скобки
    , запятая
    + знак сложения
    - знак вычитания
    * знак умножения
    / знак деления
    < знак меньше
    > знак больше
    = знак равно
    . точка
    ' апостроф
    \ обратная косая черта
    ~ тильда
    ! восклицательный знак
    # знак номера
