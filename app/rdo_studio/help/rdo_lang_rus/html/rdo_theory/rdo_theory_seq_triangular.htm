Параметры распределения: a ≤ c ≤ b, a < b

Параметр a называется нижним пределом, b - верхним пределом, c - модой (значением, встречающимся в распределении наиболее часто). В частном случае a = c или b = c треугольное распределение строится по двум точкам.

Область определения меры: [a, b]

Область применения: треугольное распределение используется для моделирования некоторой величины в условиях недостатка данных, когда точный закон неизвестен, но когда была проведена экспертная оценка области определения меры и наиболее часто встречающегося значения.
