УПРАВЛЕНИЕ ОТОБРАЖЕНИЕМ ТЕКСТА

Частично возможности по управлению отображением описаны в описании меню. В данном разделе приведем лишь функции, вызываемые с помощью нажатия комбинации клавиш.

Включение/выключение режима отображения пробелов и символов табуляции осуществляется с помощью пункта меню View->Whitespace, а также нажатием комбинации клавиш Ctrl+Shift+8.

Включение/выключение режима отображения символов конца строки осуществляется с помощью пункта меню View->End of Line, а также нажатием комбинации клавиш Ctrl+Shift+9.
