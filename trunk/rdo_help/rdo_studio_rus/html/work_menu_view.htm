МЕНЮ ПРОСМОТР

Данное меню предоставляет функции для управления отображением текста моделей.

Пункт меню Комбинация клавиш Функция
Toggle Current Fold Свернуть/развернуть текущий сворачиваемый фрагмент текста
Toggle All Folds Ctrl+Shift+0 Свернуть/развернуть все сворачиваемые фрагменты текста
Whitespace Ctrl+Shift+8 Включение/выключение режима отображения пробелов и символов табуляции
End of Line Ctrl+Shift+9 Включение/выключение режима отображения символов конца строки
Zoom In Ctrl+Num + Увеличение масштаба текста модели/графика результатов
Zoom Out Ctrl+Num - Уменьшение масштаба текста модели/графика результатов
Zoom Auto Ctrl+Num * Установление автоматического масштаба графика результатов
Reset Zoom Ctrl+Num / Возврат к первоначальному размеру текста модели/графика результатов
Workspace Alt+0 Включение/выключение режима отображения "плавающего" окна рабочего пространства
Output Alt+9 Включение/выключение режима отображения "плавающего" окна вывода
Toolbar Включение/выключение режима отображения выбранной панели
Options Вызов диалогового окна настроек программного комплекса
