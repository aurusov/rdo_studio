СТРОКА СОСТОЯНИЯ

Строка состояния имеет шесть панелей для отображения информации (см. рисунок).

В первой панели отображается текущее положение каретки (позиция в строке, номер строки) в редактируемом объекте.

Во второй панели отображается состояние редактируемого объекта. В том случае, если имело место изменение текста объекта, в данной панели отображается слово "Modified". Если объект доступен только по чтению, то на панели отображается фраза "Read only".

Третья панель – индикатор режима редактирования текста (вставка или замена) в текущем объекте. Переключение режимов осуществляется клавишей Insert. Если для данного объекта включен режим замены, то на панели отображается слово "Overwrite" и изменяется форма каретки.

Четвертая панель служит для отображения текущего модельного времени.

Пятая панель отображает состояние модели в процессе моделирования: режим максимальной производительности ("No Show"), режим анимации ("Animation") или режим паузы ("Monitor").

Шестая панель отображает скорость процесса моделирования.
