МЕНЮ INSERT

Данное меню предоставляет функции для вставки синтаксических конструкций языка и заготовок (шаблонов) для написания элементов модели. Данный пункт меню также предоставляет возможности для работы с четырьмя дополнительными текстовыми буферами.

Пункт меню Комбинация клавиш Функция
PAT, RTP, RSS, OPR, FRM, FUN, DPT, SMR, PMD При выборе данных пунктов меню открывается подменю, содержащее синтаксические конструкции и шаблоны для соответствующего объекта модели. Выбранный из подменю элемент вставляется в текст модели
Functions При выборе данного пункта меню открывается подменю, содержащее перечень стандартных арифметических и логических функций языка РДО. При выборе функции из подменю в текст модели вставляется имя выбранной функции
Buffer 1, Buffer 2, Buffer 3, Buffer 4 Работа с четырьмя текстовыми буферами (редактирование и формирование содержимого, вставка в текст модели и очистка)
