СКРЫТИЕ ФРАГМЕНТОВ ТЕКСТА

В данной версии программного комплекса существует возможность скрытия фрагментов текста. Понятие скрываемого фрагмента текста различно для различных объектов модели. Ниже дается их перечень:

Закладка главного окна Объект модели Скрываемый фрагмент
PAT Объект образцов Образец операции
RTP Объект типов ресурсов Тип ресурса
RSS Объект ресурсов Весь текст объекта ресурсов
OPR Объект операций Весь текст объекта операций
FRM Объект кадров анимации Кадр анимации
FUN Объект констант, функций и последовательностей Раздел описания констант, функция или последовательность
DPT Объект точек принятия решений Точка принятия решений
PMD Объект требуемых статистик Весь текст объекта требуемых статистик

Необходимость сворачивания фрагментов текста может возникнуть при достаточно большом количестве элементов в объекте модели. Например, при наличии большого числа образцов, удобно свернуть часть из них (те, которые не редактируются в данный момент), чтобы сосредоточить внимание на редактируемых. При сворачивании фрагмента текста от него остается первая строка, отчеркнутая горизонтальной чертой. Текст фрагмента не теряется, и фрагмент может быть в любой момент развернут.

Начало (а также, возможно, границы) сворачиваемого фрагмента текста отображается в специальном поле слева от поля редактирования текста в виде маркера.

Свернуть/развернуть текущий сворачиваемый фрагмент текста, можно выбрав пункт меню View->Toggle Current Fold или нажав левой кнопкой мыши на маркер начала сворачиваемого фрагмента в специальном поле поля редактирования (см. pис.).

Свернуть/развернуть все сворачиваемые фрагменты текста, можно выбрав пункт меню View->Toggle All Folds или нажав комбинацию клавиш Ctrl+Shift+0.
