СВОЙСТВА ГРАФИКА

В этой вкладке перечислены свойства, относящиеся к свойствам конкретного графика.

Наименование настройки Назначение
Chat series Позволяет выбрать график, настройки которого будут изменены
Serie title Название графика, которое пишется в легенде
Show serie in legend Отображать в легенде название графика
Color Цвет графика
Marker Тип маркера
Marker size Размер маркера
Draw marker Отображать маркеры на графике
Transparent marker Оставить маркер прозрачным (если маркер имеет замкнутую форму, то его можно залить цветом)
