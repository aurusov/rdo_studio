Окно рабочего пространства

Данное окно предназначено для работы со структурированной информацией различного назначения. Список закладок:

  • Tracer – отображает список трассируемых объектов в процессе моделирования, позволяет отобразить график по выбранному элементу
  • Frames – отображает список доступных окон анимации в процессе моделирования, позволяет переключиться на выбранное окно
