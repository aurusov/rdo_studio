ЗАПУСК ПРОГРАММНОГО КОМПЛЕКСА

Для начала работы системы необходимо запустить модуль RAO-studio.exe. После загрузки программы откроется основное окно. Оно имеет главное меню, кнопки панели управления, "плавающие" окна рабочего пространства и вывода со своими закладками, строку состояния и окно с набором закладок ("PAT", "RTP", "RSS", "OPR", "FRM", "FUN", "DPT", "SMR", "PMD", "PMV", "TRC"), соответствующих объектам модели для редактирования новой модели.
