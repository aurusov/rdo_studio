СОЗДАНИЕ МОДЕЛИ

Создать новую модель можно путем выбора пункта меню File->New, нажав комбинацию клавиш Ctrl+N или нажав на соответствующую кнопку на панели инструментов проекта - .

Будет создана новая модель. Если перед созданием новой модели редактировалась другая модель, и изменения не были сохранены, то выдается запрос на сохранение изменений. Во вновь созданную модель вставляются шаблоны для создания объектов модели.
