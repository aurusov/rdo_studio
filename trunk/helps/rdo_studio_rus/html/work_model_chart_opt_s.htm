СВОЙСТВА ГРАФИКА

В этой вкладке перечислены свойства, относящиеся к свойствам конкретного графика.

  • Chat series – Отображает название параметра
  • Serie title – Название графика, которое пишется в легенде графика.
  • Show serie in legend – отображать в легенде название графика.
  • Color – Цвет графика.
  • Marker – Тип маркера
  • Marker size – Размер маркера
  • Draw marker – Отображать маркеры не графике.
  • Transparent marker – Заливать содержимое маркера (если маркер имеет замкнутую форму).
