АВТОМАТИЧЕСКОЕ ПОСТРОЕНИЕ ГРАФИКОВ

В данной версии программного комплекса существует возможность масштабирования текста модели.

В данной версии программного комплекса имеется возможность визуально отображать процессы происходящие в модели в виде графиков. Добавление графиков состояния ресурса становится возможным только после запуска модели. Графики состояния ресурса можно добавлять как в процессе работы модели в режиме анимации, так и после окончания работы модели. График состояния ресурса может быть отображен только в том случае, если конкретный ресурс трассируется.
Для добавления графика, во вкладке Tracer окна Workspace в дереве модели найдите необходимый вам параметр ресурса . Далее, чтобы создать новое окно графика нужно щелкнуть два раза по выбранному параметру ресурса левой кнопкой мыши или один раз правой и в выпадающем меню выбрать Add To New Chart. После этого отображается график, причем по оси абсцисс откладывается время происхождения событий, при котором изменяется значение параметра.

При первом построении графика масштаб по оси ординат устанавливается автоматически. Для изменения масштаба по оси абсцисс можно воспользоваться панелью инструментов Zoom Toolbar или использовать выпадающее меню при правом щелчке мышки по области графика (см. Рис. 9). Основными функциями масштабирования являются:
Пункт меню Сочетание клавиш Действие
Zoom In Ctrl+Num- Увеличение
Zoom Out Ctrl+Num- Уменбшение
Zoom Auto Ctrl+Num* Автоматический подбор масштаба по ширине области графика
Reset Zoom Ctrl+Num/ Восстановление начального масштаба

Существует так же возможность добавлять на уже существующий график другие. Такая возможность осуществляется перетаскиванием мышкой параметра ресурса на область уже построенного графика.
Основные свойства графика можно изменить в пункте меню Chart->Options или выбрав пункт Options в описанном выше выпадающем меню.
